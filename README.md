# Book-project
Simple backend for storing books and witters.


# Commands

To start first time

```
npm install
```
to install all the packages.


After the installation run 
```
npm run server
```

#Important for .env

Your .env should have the following:
NODE_ENV = development
PORT = 8000
MONGO_URL = your database link
JWT_SECRET = your seed
